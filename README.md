## 🧩 What promptref does

promptref helps you keep track of LLM prompts in a simple, Git-like way. You can save changes, compare versions, create branches, and roll back to older prompts from the command line.

It is made for people who work with prompts in tools like OpenAI and Anthropic. If you change prompts often and want a clean history, promptref gives you one place to manage them.

## 🌿 Branches and Versions

promptref uses the same ideas as Git, but for prompts.

### Versions
Each save gives you a new point in time. You can see how your prompt changed over days or weeks.

### Branches
Use branches when you want to test a new prompt idea without changing the main version. This is useful when you want to try:

- A shorter prompt
- A different system message
- A new style guide
- A prompt for another model

### Rollback
If a change hurts the result, you can return to an older version. This keeps you from losing a prompt that worked well.

## 🔍 Diff and Compare

The diff view shows what changed between two prompt versions. This helps you answer questions like:

- Did I remove an important rule?
- Did I change the tone?
- Did I add extra words?
- Did a test prompt drift from the original?

This is useful when you update prompts often and need to know why output changed.
